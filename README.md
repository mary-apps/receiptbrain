# ReceiptBrain 🧾

**Scan. Extract. Done.**

ReceiptBrain uses AI to scan receipts and extract expenses instantly. Get a visual spending dashboard, auto-categorization, and tax-ready CSV exports.

## Features

- 📸 **Smart Scanner** — Point camera at receipt, AI extracts all data
- 📊 **Expense Dashboard** — Beautiful charts showing spending trends
- 🏷️ **Auto-Categorization** — AI sorts expenses by category
- 🔍 **Searchable Archive** — Find any receipt by merchant, date, or amount
- 📤 **CSV Export** — Tax-ready reports in one tap
- ✨ **Scanning Overlay** — Real-time camera overlay with extraction feedback

## Tech Stack

Flutter (iOS) • Riverpod • GoRouter • Camera • fl_chart • Lucide Icons • shimmer • flutter_animate
